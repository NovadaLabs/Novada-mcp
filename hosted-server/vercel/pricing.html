[ RULES ]

HOW USAGE IS COUNTED 用量如何计算

Two separate meters: the free-gateway call cap (1 per call) and your Novada balance (per-call usage, proportional to work done). 两套独立计量:免费网关调用上限(每次调用计 1 次)与 Novada 余额(按实际用量扣费)。

1

GATEWAY CAP: 1 PER CALL 网关上限:每次调用计 1

Each successful MCP tool call counts as 1 against the free-gateway monthly cap, regardless of tool. This is only the anti-abuse counter — your actual cost is drawn from your Novada balance (Rule 3), where some tools do more upstream work. 每次成功的 MCP 工具调用在免费网关月度上限中计为 1 次,与具体工具无关。这只是防滥用计数器 — 实际费用从 Novada 余额扣除(见规则 3),部分工具会发起更多上游工作。

2

CAP = PER TOKEN, MONTHLY 上限按 TOKEN 每月计

The free-gateway cap is attached to your API key, not an IP or device. The same key works across Claude Desktop, Claude Code, Cursor, and local npx novada-mcp — but the cap applies only to the hosted gateway; the local server has no monthly gateway cap. 免费网关上限绑定在你的 API 密钥上,与 IP 或设备无关。同一密钥可用于 Claude Desktop、Claude Code、Cursor 和本地 npx novada-mcp — 但该上限仅作用于托管网关;本地服务不受月度网关上限限制。

3

BALANCE: PAY-AS-YOU-GO 余额:按用量付费

Each call draws per-call usage from your Novada balance at standard rates. Some tools make multiple upstream calls and draw proportionally — e.g. novada_research reads several sources and novada_crawl fetches many pages in one call. Failed calls don't count against the free-gateway cap (it's refunded), though any upstream work already done may still draw balance. 每次调用按标准费率从 Novada 余额扣除用量。部分工具会发起多次上游调用并按比例扣费 — 例如 novada_research 会读取多个来源、novada_crawl 单次调用抓取多页。失败调用不计入免费网关上限(会退还),但已完成的上游工作仍可能扣除余额。

4

RATE: 60 REQ / MIN / IP 速率:每 IP 60 次/分钟

The hosted endpoint enforces 60 requests per minute per source IP, independent of the gateway cap. Requests that exceed the rate limit receive 429 Too Many Requests and are not counted against your free-gateway cap. (This 429 is the only case the server returns 429 — reaching the monthly gateway cap returns a normal tool-error result, not 429.) 托管端对每个来源 IP 每分钟限制 60 次请求，与免费网关上限独立计算。超出速率限制的请求返回 429 Too Many Requests，不计入免费网关上限。(这是服务器唯一返回 429 的情形 — 达到月度网关上限时返回的是普通工具错误结果,而非 429。)
